# Deep-Learning-Project
This project explores whether deep learning models trained on EEG data can distinguish human consciousness states (wake, REM, NREM) and learn representations that align with Integrated Information Theory (IIT) using information-theoretic measures like mutual information and correlation.

## Data Availability

- The original dataset was sourced from:  https://physionet.org/content/sleep-edfx/1.0.0/
- The processed dataset used in this study is available at:  https://www.kaggle.com/datasets/aurora1620/sleep-psg-processed-dataset-npz-hypnograms
